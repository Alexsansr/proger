Какой ЯП выбрать в 2020?

Этот вопрос мучает начинающих и заядлых программистов. В действительности, а какой выбрать?

Ответ зависит от того чем вы планируете заниматься. Это может быть WEB разработка, игры или десктопные программы, а ещё этот выбор зависит от платформы. Например:

Android, Windows, Linux … . В этой статье вы разберётесь с выбором ЯП.

 

 

 

1) C#

Csharp создан компанией Microsoft в 2001 году. Он отлично подходит к платформе Windows.

Преимущества:

·        Десктопные программы на Windows

·        Можно писать Android программы благодаря Xamarin

·        Игры

·        Типы данных имеют фиксированный размер(Вы всегда будете знать сколько оперативки потребляет ваша программа)

·        Бесплатный (для небольших компаний и групп людей)

·        Хорошая документация

·        Автоматический сбор мусора

 

 

 

Недостатки:

·        Для больших компаний C# обойдётся в большую сумму

·        Главная ориентация платформа Windows

·        C# легко дизассемблируется (Вашу программу могут изменить)

 

 

2) Java

Java был создан компанией Sun MicroSystems 23 мая 1995 г. и выкуплен компанией Oracle.

Java программы транслируются в байт-код Java и воспроизводятся (JVM) виртуальной машиной Java. По сути Java интерплитируемый ЯП. Программа на java запустится на любой платформе у которой установлена JVM.

Преимущества:

·        Java запустится на любой платформе

·        Хорошая документация

·        Высокого уровня с C-подобподобым

·        Безопасный

·        Автоматический сбор мусора

·        Бесплатен (если вы не собираетесь продавать программу)

 

 

 

 

Недостатки:

·        Платен (если вы собрались продавать программу)

·        Низкая производительность

 

 

 

3) C++

 

Язык C++ появился на свет в 1980 году Бьёрн Страуструп придумал дополнения к языку C для себя. Несмотря на возраст языка его используют до сих пор.

 

 

 

Преимущества:

·        Минимальные требования для запуска программы.

·        Быстрая компиляция и запуск

·        Поддерживает множество платформ

·        Есть возможность управления памятью

·        Бесплатен

 

 

 

 

Недостатки:

·        Сложный для начинающих

·        Нет автоматической сборки мусора

·        При использовании модуля медленнее компилируется

 

 

 

 

4) Python

Python начал разрабатываться в конце 1980 г. Гвидо ван Россумом. Python подойдёт для новичков. На Python существует много библиотек (Дополнений), так что скучно не будет.

 

 

 

 

Преимущества:

·        Многплатформенность

·        Хорошая документация

·        Большое сообщество

·        Легко и быстро писать код

·        Легко расширяется

·        Встроенный авто-установщик модулей (PIP)

·        Каждый день появляются новые модули

·        Тип переменных определяется автоматически\

·        На нём можно написать всё: сервер, десктоп программа, игры и т.д.

·        Бесплатен

 

 

 

Недостатки:

·        Медленный (По сравнению с компилируемыми ЯП)

 

 

 

 

 

 

5) Ruby

Ruby был создан в 1993 году  Юкихиро Мацумото. Его можно использовать для Web разработки и не только. У него C-подобный синтаксис.

 

 

Преимущества:

·        Подходит ко многим платформам

·        Можно использовать в Html

·        Присутствует встроенный отладчик

·        Продвинутые способы работы со строками

·        Есть инструменты для увеличения безопасности

 

 

 

 

Недостатки:

·        Для начинающего может показаться сложным

·        Мало документирован

·        Медленный

·        Редко совершенствуется

 

 

 

 

 

6) PHP

 

PhP изобрели 1995 г. Раумусом Лердорфом. В 1994 он создал инструмент на C, который вёл учёт посетителей на его странице. Ему не хватало функционала и он создал PHP.

 

 

Преимущества:

·         Простой и понятный

·         Высокая скорость работы

·         Многоплатформенный

·         Можно разработать отдельно от Html, а потом соединить

·         Поддерживается большинством хостингов

·         Большое кол-во библиотек (Дополнений)

 

 

Недостатки:

·         Безопасность - это не про него

·         Слабо работает с исключениями

 

 

 

7) JavaScript

 

 

 

JavaScript был создан 1995 г. Бренданом Эйхом. Он внедрил что-то похожее на Scheme в JavaScript. JavaScript в основном используют для дополнения Html страницы функционалом, но его можно использовать как интерплитатор.

 

 

 

Преимущества:

·         Изменяет Html страницы

·         Может работать с Cookie-файлами

·         Вносить изменения в свой код

·         Добавлять/удалять теги Html документа

·         Изменять стиль Html документа

 

 

 

 

Недостатки:

 

·         Ваш код полностью доступен всем пользователям

·         Существуют непонятные, но исправляемые ошибки

 

 

7) HTML

 

Html был изобретён в 1986-1991г. Тимом Бернерсом-Ли. Изначально его использовали для обмена научной информацией, но его теперь использует каждый для создания сайта.

 

 

 

 

 Преимущества:

·         Подходит для создания корпуса веб страницы

 

 

Недостатки:

·         Подходит только для вёрстки сайтов
